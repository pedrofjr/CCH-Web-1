Seus projetos

Conversor D2B


O Conversor D2B (Decimal em Binário) foi um projeto desenvolvido na disciplina de Estrutura de Dados no curso de Engenharia de software.

Nele, você fornece um número inteiro positivo e ele deve retornar o número convertido em binário. Dessa forma, o objetivo era explorar o uso de pilhas estáticas na linguagem de programação C.

Compilador


O Compilador foi um projeto desenvolvido na disciplina de Estrutura de Dados no curso de Engenharia de software.

Nele, você fornece uma operação matemática, e ele deve retornar as instruções de como seriam processados esses dados em uma calculadora em baixo nível. Dessa forma, o objetivo era explorar o uso de pilhas dinâmicas na linguagem de programação C.

Lista de Pacientes


A Lista de Pacientes de Clínica, foi um projeto desenvolvido na disciplina de Estrutura de Dados no curso de Engenharia de software.

Nele, você fornece uma lista de pacientes, com seus respectivos dados, e ele deve retornar a lista em ordem crescente, decrescente ou apenas 1 elemento, dependendo da entrada. Dessa forma, o objetivo era explorar o uso de listas duplo encadeadas circulares na linguagem de programação C.
